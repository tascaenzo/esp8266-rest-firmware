ESP8266 GPIO Automation Firmware

Secure, deterministic REST firmware for ESP8266-based devices.

This firmware exposes a JSON-based REST API to control GPIOs, schedule actions and inspect device state, while remaining suitable for low-memory embedded environments.

Design Goals

  • Predictable behavior
  • Replay-safe authentication
  • No TLS required
  • Bounded memory usage

What you can do here

  • Inspect live device state
  • Experiment with authenticated API calls
  • Study request signing and payload rules
  • Debug GPIO and cron behavior
